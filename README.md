# О проекте:
Веб-приложение для поиска книг по названию, автору или ISBN. ЛКМ для перехода на карточку книги.
В карточке книги указано название, автор, ISBN (если есть), год выпуска, цена, жанр и кликабельные ссылки на сайты онлайн-магазинов.
Для запуска проекта введите в консоли команду ```uv run python manage.py runserver``` (убедитесь, что у вас установлена библиотека uv для Python).
